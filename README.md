### Getting Started

Prerequisites: Node.js & npm installed.

1.  Clone the repository:
    ```sh
    git clone <YOUR_GIT_URL>
    ```
2.  Navigate to the project directory:
    ```sh
    cd first-step
    ```
3.  Install dependencies:
    ```sh
    npm install
    ```
4.  Start the development server:
    ```sh
    npm run dev
    ```
